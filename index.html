Connect Spotify

Earworm plays music clips directly in this browser tab using the Spotify Web Playback SDK. No phone or external device needed — audio comes straight from this page.

Enter your Spotify app's Client ID below, then hit Connect. Your Client ID is never sent anywhere except directly to Spotify.

→ Get your Client ID from the Spotify Developer Dashboard

Setup Guide

1. Go to developer.spotify.com/dashboard
2. Create an app (any name, any description)
3. In the app settings, add this page's URL as a Redirect URI:
   
4. Copy your Client ID and paste it above
5. Make sure your browser volume is on — audio plays directly here!
